# 🎬 Movie Explorer

A modern and elegant web app built with **Next.js + TypeScript + Tailwind CSS** that allows users to search for movies and TV shows, view detailed information, and enjoy a seamless browsing experience with dark/light mode support.

---

## 🚀 Features

- 🔍 Search movies
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎯 Detailed movie information (rating, genres, release date, synopsis)
- 💾 Search persistence when navigating back
- ⚡ Smooth animations and micro-interactions

---

## 🛠 Tech Stack

- Next.js
- TypeScript (Type safety)
- Tailwind CSS v4 (Modern styling)
- Heroicons (Beautiful icons)

---

## 📡 API Used

TVmaze API (No API key required):
https://api.tvmaze.com/search/shows?q={query}
https://api.tvmaze.com/shows/{id}


---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Qasem-21/movie-explorer-nextjs-typescript-tailwind.git
```
--- 

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run the development server

```bash
npm run dev
```
---

### 4. Open in browser

```
http://localhost:3000
```
